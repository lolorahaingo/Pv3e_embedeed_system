# Embedded-Systems (PV3e) - SerialPlotter

Simple serial plotter with electron.

## Requirements

* *nodejs* **v11.15.0+**

## Getting started

* Clone it

```bash
git clone --branch PV3e https://github.com/raymas/Embedded-Systems/SerialPlotter.git
```

* Intall it.

```bash
cd SerialPlotter
npm install
```

* Plug the receiver dongle.

* Start it.

```bash
npm start
```

* Done
